Bentö



Technology: Rails, React, JavaScript, AWS S3


August 2017

Bentö is a web-app clone of the popular design and artwork showcase platform Behance. It utilizes Ruby on Rails on the backend, a PostgreSQL database, and React.JS with a Redux architectural framework on the frontend. Amazon Web Services stores project images and user avatars.
